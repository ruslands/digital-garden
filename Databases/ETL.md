Some popular no-code tools for building ETL pipelines:

1. Zapier - allows you to connect different applications and automate data transfer between them.
2. Airflow - an open-source platform to programmatically author, schedule, and monitor workflows.
3. AWS Glue - a fully-managed ETL service that makes it easy to move data between data stores.
4. Google Cloud Dataflow - a fully-managed, serverless data processing service for building and executing Apache Beam pipelines.
5. Talend - a cloud-based integration platform that offers no-code options for ETL pipeline development.
6. Tray.io - a cloud-based automation platform that offers a visual workflow editor to build and automate data pipelines.
7. Workato - a cloud-based integration platform that allows users to create custom integrations and automations between different applications.
8. Hevo - a no-code data pipeline platform that allows users to integrate data from multiple sources and load it into a data warehouse.
9. Apache NiFi - an open-source data integration platform that provides web-based flow editing, real-time analytics, and data provenance.
10. Blendo - a cloud-based data integration platform that allows users to extract data from various sources and load it into data warehouses or other destinations.
